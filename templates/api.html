The following examples will help you understand how the API will be used. As of now, the API calls should be directed to their respective Heroku URLs. The domains would be redirected to a single URL if used as a legitimate product. Below is a series of examples on how to make the calls.

curl -X GET 'https://lmsapi-student.herokuapp.com/?key=<apikey>'
curl -X GET 'https://lmsapi-student.herokuapp.com/students?key=<apikey>'
curl -X POST 'https://lmsapi-student.herokuapp.com/students?key=<apikey>&firstname=<firstname>&lastname=<lastname>&email=<email>'
curl -X PUT 'https://lmsapi-student.herokuapp.com/students/<email>?key=<apikey>&firstname=<firstname>&lastname=<lastname>&email=<email>'
curl -X DELETE 'https://lmsapi-student.herokuapp.com/students/<email>?key=<apikey>'

Students Service

URL: https://lmsapi-student.herokuapp.com/

/students
Method Action Arguments
GET Retrieves all of the students under that user None
POST Adds a student firstname, lastname, email
/students/<student_email>
Method Action Arguments
PUT Updates a student firstname, lastname
DELETE Removes a student None

Courses Service

URL: https://lmsapi-course.herokuapp.com/

/courses
Method Action Arguments
GET Retrieves all of the courses under that user None
POST Adds a course number, title, prereq
/courses/<course_number>
Method Action Arguments
PUT Updates a course title, prereq
DELETE Removes a course None
/courses/<course_number>/student
Method Action Arguments
GET Gets all students in a course None
POST Adds a student to a course email, enrolled
PUT Updates a students enrollment email, enrolled
DELETE Removes a student from a course email

Assignments Service

URL: https://lmsapi-assignment.herokuapp.com/

/assignments
Method Action Arguments
POST Adds an assignment to a course number, course_num, name, type, due
/assigments/<course_number>
Method Action Arguments
GET Gets all assignments in a course None
PUT Updates an assignment in a course number, name, type, due
DELETE Removes an assignment from a course number
/assignments/<course_number>/<assignment_number>
Method Action Arguments
GET Gets all students with an assignment None
POST Adds a student to an assignment email
PUT Updates a students status in an assignment email, grade
DELETE Removes a student from an assignment email
